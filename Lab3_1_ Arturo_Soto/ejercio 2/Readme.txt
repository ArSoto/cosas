Lab 3,  16/10/2018

El programa recibe nombre entregados por el usuario y los ordena alfabeticamente

Para ejecutar el programa debe tener instalado los compiladores
de c++ y estar en un entorno Linux.


Para iniciar el programa primero inicie el terminal y diríjase a la carpeta
donde se encuentra el codigo y digite los siguentes comandos:
	~make
	~/. main

Este programa esta diseñado para sistemas Linux y no se garantiza un
optimo desempeño en otros entornos.
